# Macro-Reports

Monthly macroeconomic research notes analyzing U.S. labor market and inflation data using primary economic releases (ADP, BLS, FRED). Focus is on sector rotation, labor dynamics, and cyclical positioning.

---

## Current Report
- ADP Employment – May 2026

---

## Framework
Each report follows:
- Data overview
- Establishment size breakdown
- Sector breakdown
- Month-over-month comparison
- Macro interpretation

---

## Sources
- ADP National Employment Report
- Bureau of Labor Statistics (when applicable)
- Federal Reserve Economic Data (FRED)

---

## Structure Goal
Build a consistent macro research archive with monthly labor and inflation reports.

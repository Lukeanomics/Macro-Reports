## Report Viewing Notice

GitHub's built-in PDF viewer may not display the entire report and can occasionally stop rendering later pages.

If a PDF appears incomplete:

* Download the PDF and view it locally for the full report.
* Alternatively, read the corresponding `.md` version, which is fully viewable on GitHub.

Each report is provided in both formats:

* `.md` — GitHub-friendly reading version
* `.pdf` — Archived/export version

The PDF file itself is complete even if GitHub's preview does not display every page.
